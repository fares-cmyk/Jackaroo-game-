
# Jackaroo Game — GUC Computer Programming Lab

A single-player digital adaptation of the classic Middle Eastern board game **Jackaroo**

## Overview
The player competes against 3 CPU opponents, moving 4 marbles from their Home Zone to their Safe Zone across a 100-cell track. The game uses a custom 102-card deck with unique actions to move, swap, burn, and save marbles strategically.

## Features
- 1 human player vs 3 CPU opponents
- 100-cell circular track with Home, Base, and Safe Zones
- 15 card types including special Wild Cards (Burner & Saver)
- Trap cells, marble collisions, and strategic blocking mechanics
- Card actions: movement, swapping, fielding, burning, saving, and discarding

## How to Win
Be the first player to move all 4 marbles into your Safe Zone.

## Requirements
- Cards.csv file must be present in the project directory
